# chrome-gpu
在 Google Chrome 打开 chrome://gpu 检查.<br>
<br>
如果发现 Graphics Feature 没全被打开,<br>
你可以下载这个 BASH 运行..<br>
<br>
$ wget https://github.com/yomun/chrome-gpu/raw/master/chrome.sh<br>
$ sudo bash chrome.sh<br>
<br>
然后再检查 chrome://gpu, 功能应该全打开了<br>
